## 🚀 How to Use This Repository

### For Evaluation
Each project can be accessed directly via its folder. Click on the project link above to view:
- Project description
- Jupyter notebooks
- Analysis reports (PDF)
- Source code and data files

### Running Projects Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SholaSaliu/3mtt-dhfoundation-projects.git
   cd 3mtt-dhfoundation-projects
   ```

2. **Navigate to a project:**
   ```bash
   cd 01-regression-analysis
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Open Jupyter Notebook:**
   ```bash
   jupyter notebook notebooks/
   ```
